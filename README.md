# 🌴 万宁之旅 - 移动端行程规划

一个专为移动端优化的万宁旅游行程网页，方便在手机上随时查看。

## 📱 在线预览

👉 [https://m1234532.github.io/wanning-trip/](https://m1234532.github.io/wanning-trip/)

## 🗓️ 行程概览

- **Day 1 (6月19日 周五)**: 抵达三亚 → 自驾石梅湾 → 日月湾日落 → 216餐吧
- **Day 2 (6月20日 周六)**: 洲仔岛桨板浮潜 → 日月湾冲浪 → 泰式按摩 → 椰子鸡
- **Day 3 (6月21日 周日)**: 椰林公路 → 山钦湾 → 海口骑楼老街 → 海大南门夜市
- **Day 4 (6月22日 周一)**: 海口 → 返程上海

## ✨ 功能特点

- 📱 **移动端优先**：完美适配手机屏幕
- 🎨 **精美设计**：渐变背景 + 卡片式布局 + 时间轴
- 📅 **日期切换**：顶部标签快速切换每一天
- 🏷️ **活动分类**：航班、车程、美食、住宿、活动标签一目了然
- ⚡ **离线可用**：单文件 HTML，无需网络即可打开

## 🚀 使用方式

1. **手机浏览器访问**：直接打开在线链接
2. **添加到主屏**：在 Safari/Chrome 中选择"添加到主屏幕"，像 App 一样使用
3. **本地查看**：下载 `index.html` 文件，离线也能浏览

## 🛠️ 技术栈

- 纯 HTML + CSS + JavaScript
- 响应式设计（Mobile First）
- CSS 动画 + 毛玻璃效果
- 无需任何框架或依赖
